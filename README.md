# 🎥 StreamFlix

Video streaming platformu. Netflix/YouTube tarzı.

## Özellikler
- 🎬 Video streaming (HLS/DASH)
- 📺 Canlı yayın
- 💬 Canlı sohbet
- 📊 İzleme analitikleri
- 🔔 Abonelik sistemi
- 📱 Responsive player

## Teknolojiler
- React + Video.js
- Node.js + Express
- MongoDB
- AWS MediaConvert
- CDN

## Kurulum
```bash
npm install
npm run dev
```
